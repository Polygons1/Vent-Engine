### Vent-Window

A Window creation & management Library

## Supported Platforms

- Desktop
    - [Windows](#windows)
    - [macOS](#macos-planned)
    - [Unix/Wayland](#unix-wayland)
    - [Unix/X11 Info](#unix-x11)
- Mobile
    - [Android](#android-planned)
    - [IOS](#ios-planned)
